Introduced: 2025-10-21. Written into `_posts/2025-10-21-CELPIP.md` under "### Part 6 - Dealing with a Difficult Situation".

**Why:** Learner prefers formula-like templates. This 8-slot formula covers all band 7-8 requirements: task fulfillment (opener + position + solution), coherence (logical slot order), vocabulary (expression bank provided), and listenability (60-second time budget per slot).

## The 8-slot formula (reference only — full template in post)

```
[OPENER] → [CONTEXT] → [YOUR_POSITION] → [REASON_1+DETAIL]
→ [CONCESSION] → [REASON_2/SOFT_SOLUTION] → [CLOSER]
```

## Timing budget (60 seconds)
- 0–10s: OPENER + CONTEXT
- 10–18s: YOUR_POSITION
- 18–35s: REASON_1 + DETAIL
- 35–44s: CONCESSION
- 44–55s: REASON_2 / SOFT_SOLUTION
- 55–60s: CLOSER

## Decision rule taught
- IF you can defend one side with reason + evidence → choose that side
- IF both sides equal → prefer "advocate for someone" (easier to fill 60s)
