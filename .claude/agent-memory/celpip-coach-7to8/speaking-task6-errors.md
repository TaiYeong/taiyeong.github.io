Session date: 2025-10-21. Task: Speaking Task 6 (Dealing with a Difficult Situation — bakery/Tom scenario). Learner chose to address the manager.

**Why:** These errors reveal patterns that recur under time pressure. Tracking them allows targeted drilling in future sessions rather than re-diagnosing from scratch.

**How to apply:** In future Speaking or Writing sessions, watch for these exact patterns and call them out by rule name immediately.

## Confirmed error patterns

| Error | Rule | Fix |
|---|---|---|
| `be + agree` ("you are agree") | `agree` is a verb, not an adjective. Never use `be + bare verb`. | "I hope you agree" / "I hope you will consider" |
| Sentence-initial `Because` (fragment) | `Because` is a subordinating conjunction — must connect two clauses. | Lead with main clause: "This is because..." |
| `any improvement` in positive statement | `any` = negatives/questions. Positive prediction needs `some` or adjective. | "some improvement" / "significant improvement" |
| `the hard time has been gone` | `gone` = moved away. For a period ending, use `be over` or `have passed`. | "his difficult time has passed" / "things are better now" |
| `It would be good to believe that` | Avoids committing to an opinion. CELPIP rewards direct stance. | "I am confident that" / "I firmly believe that" |
| Missing concrete next step | Task 6 band 7+ requires a soft solution / next action, not just advocacy. | Add [SOFT_SOLUTION] slot: "What I'd suggest is..." |
