taiyeong.song is a VFX pipeline TD (Technical Director) at Bardel (or similar VFX studio). They maintain a personal knowledge-base blog at taiyeong.github.io covering pipeline tooling, USD, Houdini, Katana, Blender, CMake, Docker, Python, and AI tools.

Blog posts are bilingual (Korean explanations, English section headings and code). The author writes with a practitioner's directness — no fluff, no hedging. Korean prose is used for conceptual explanations; English is used for code, headings, and technical terms.

They are actively building a Blender pipeline (Surfacing, Lighting, Rendering depts) and investing in Claude Code / AI tooling for pipeline research and automation. See [[project-blender-pipeline]] and [[project-claude-tooling]] for active project context.
